# Rails 7 Demo

This project was created with the official Ruby on Rails [video tutorial](https://rubyonrails.org/).

## Installation

1. Clone this repo:
  ```
  git clone git@github.com:yesnik/rails7-demo.git
  ```
2. Install dependencies:
  ```
  cd rails7-demo
  bundle install
  ```
3. Run setup command:
  ```
  rails db:setup
  ```
4. Run development server:
  ```
  rails server
  ```
